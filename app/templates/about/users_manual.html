User's Manual

HRMS User’s Manual

Welcome! This guide shows you how to use the HRMS app in simple steps—whether you’re filing leave, managing employees, uploading DTR logs, or generating payroll reports.

Tip: If you ever get stuck, open this page again from ABOUT → User’s Manual. You can also use Change password under your name at the top-right.

Quick start

  1. Log in using your assigned Employee ID and password.
  2. Check your dashboard for shortcuts and status counts.
  3. Do your task (apply leave, upload DTR, manage employees, etc.).
  4. Log out when you’re done—especially on shared computers.

Navigation & menus

Your menu may look slightly different depending on your role (Admin/HR, Manager, Employee, DTR Uploader, Payroll Maker).

Dashboard

Home screen with quick stats and recent activity.

Settings (Admin/HR)

Manage users, departments, positions, salary grades, and status of appointment.

Employee Management

View employees, add new employees, edit details, and open PDS records.

Daily Time Records (DTR)

View DTR records, upload logs (if allowed), and manage justifications.

Leave Management

Apply for leave, view leave history, approve leave (Managers/Admin).

Reports (Payroll Maker/Admin/HR)

Generate payroll summary and view submitted payroll history.

Common tasks (step-by-step)

A) Change your password

  1. Click your name at the top-right.
  2. Select Change password.
  3. Enter your current password and your new password.
  4. Click Change password.

B) Add a new employee (Admin/HR)

  1. Go to Employee Management → Employees.
  2. Click + Add Employee.
  3. The Employee ID is automatically generated. Fill in the rest of the form.
  4. Click Add Employee.

Note: If you see a duplicate/invalid ID, refresh the page and try again.

C) Open and update an employee PDS

  1. Open Employees list.
  2. Click an employee row to open the PDS.
  3. Update fields as needed and save.

D) Apply for leave

  1. Open Leave Management → Apply for Leave.
  2. Select the leave type and dates.
  3. Add details (reason, where required).
  4. Submit your application.
  5. Check status in Leave Management → View Leave.

E) Approve leave (Manager/Admin)

  1. Open Leave Management → Leave Approvals.
  2. Review the request details carefully.
  3. Approve or reject (when available).

F) Upload DTR log (DTR Uploader / authorized roles)

  1. Open Daily Time Records → Upload DTR Log.
  2. Pick the correct quincena.
  3. Select the .dat file and upload.
  4. Review parsed rows before saving.

G) Submit DTR justification

  1. Open Daily Time Records → DTR Justifications.
  2. Choose the date, add a clear reason, and attach proof if needed.
  3. Submit and monitor the status.

H) Generate payroll summary (Payroll Maker/Admin/HR)

  1. Open Reports → Payroll Summary Report.
  2. Select mode (quincena/month), month, year, and quincena if applicable.
  3. Click Generate List.
  4. Use Export CSV or Export PDF if needed.

What each role can do

  • Admin: Full access (settings, employees, leave approvals, reports).
  • HR: Similar to Admin for HR operations (users/departments/employees/leave settings/reports).
  • Manager: Can review/approve leave within scope and view relevant employee data.
  • Employee: Applies leave, views own leave, views personal dashboard/PDS (if enabled).
  • DTR Uploader: Uploads DTR logs and may view DTR-related pages.
  • Payroll Maker: Access to payroll reports and submission history.

Best practices

  • Be consistent: Use complete names and correct departments to keep reports accurate.
  • Double-check dates: Leave and DTR are date-sensitive; wrong dates cause payroll issues.
  • Attach proof when needed: Justifications are faster to review with supporting files.
  • Use search: Most lists support searching—use it instead of scrolling.
  • Log out: Always log out after using the system on shared devices.

Troubleshooting

Login problems

  • “Invalid credentials”: Check your Employee ID and password. Make sure Caps Lock is off.
  • Forgot password: Ask Admin/HR to reset your account password.
  • Stuck on login: Refresh the page. If it persists, restart the browser and try again.

Pages not showing / missing menus

  • Your role may not have permission for that page. Confirm your role with Admin/HR.
  • Try logging out and logging back in after role changes.

DTR upload issues

  • Wrong quincena: Select the correct quincena before uploading.
  • Missing dates warning: Upload a complete file for the selected range before saving.
  • Employee ID not found: Make sure the employee exists and the ID in the file matches the employee record.

Leave printing / export issues

  • Some print/export options may only work when a leave request is Approved.
  • If PDF export looks clipped, try browser print settings: set paper to A4 and margins to “Default”.

If something looks wrong

  • Refresh the page first (sometimes data updates after saving).
  • Check if you’re on the correct page/menu.
  • If the error message mentions “database” or “constraint”, contact Admin/HR and include a screenshot of the message.
